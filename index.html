About Me

How it all started?

I am a student specializing in web development, and I began my learning journey through online resources. By watching experienced developers code and following along, I developed a strong foundation in building websites and applications. I enjoy exploring new syntax, experimenting with different approaches, and continuously improving my skills by learning better and more efficient ways to solve problems.

Why coding?

From a young age, I was fascinated by how the web functions and how websites are built. This curiosity led me to explore coding, and over time, I committed myself to deepening my understanding of programming and web development.

How far with coding?

My coding journey has not always been easy. There were times when I had to pause or struggled with motivation, especially because I did not have friends who shared the same interest in programming. Despite these challenges, I remained committed and continued learning. I began my journey with Python, where I learned the fundamentals by building simple projects such as a calculator. Enjoying the language and its logic, I decided to expand my skills by learning a language used to build websites. This led me to HTML, where I learned how to structure web pages using headings, titles, links, and semantic tags. Before long, I was writing my own code and creating websites. To improve the visual design of my projects, I then started learning CSS to style and enhance my websites. Currently, I am learning JavaScript and have reached an intermediate level. I have a solid understanding of its syntax and functionality and am able to explain how JavaScript works and how it is used to add interactivity to web applications.
